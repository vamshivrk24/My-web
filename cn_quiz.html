Computer Network Management Quiz

  1. Which of the following is a Class A broadcast?
    1. 11.255.255.255
    2. 127.75.255.255
    3. 193.255.255.255
    4. 14.25.255.255
  2. ROM in a router is nothing like ROM in a PC. True or False?
    1. True
    2. False
  3. On a network, computers must use a common ___________ in order for communication to occur.
    1. protocol
    2. operating system
    3. manufacturer
    4. hardware platform
  4. Communications on a network originate at the _____________.

  5. A router that has eight ports will require how many IP addresses?

    2. It will be dropped.

    4. ICMP

    2. 49152 - 65535
